文档搭建¶

安装¶

pip install mkdocs

创建项目¶

mkdocs new  .
  • mkdocs.yml: 配置文件
  • docs: 文档源文件

启动服务¶

mkdocs serve

添加文章¶

  • 创建导航
nav:
    - 主页: 'index.md'
    - '云原生':
        - '基于kubeadm安装kubernetes集群': 'k8s/install_docker_k8s.md'

建立网站¶

 mkdocs build

部署文档¶

mkdocs gh-deploy
